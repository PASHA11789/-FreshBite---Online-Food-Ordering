================================================================================
                    FRESHBITE ONLINE FOOD ORDERING SIMULATION
                               PROJECT REPORT
================================================================================

1. PROJECT OVERVIEW
--------------------------------------------------------------------------------
FreshBite is a premium, highly responsive online food ordering front-end 
application built with React and Tailwind CSS. The system models a real-world
customer journey, including browsing mock restaurants, sorting and filtering menus 
by culinary and dietary options, reviewing meal details, managing an interactive 
cart drawer, completing a multi-option checkout form, and tracking simulated 
active orders in real time.

2. CORE CONCEPT & SCOPE
--------------------------------------------------------------------------------
The project delivers a fully simulated food-delivery platform experience without 
requiring external database connections or active payment APIs. Instead, state is 
managed via React Context, and persistent structures (such as user session keys, 
wishlists, saved addresses, and simulated orders) are saved directly into the 
browser's Local Storage. 

Key objectives accomplished in the application include:
- Clean modular component structure with reusable layouts.
- Absolute design responsiveness across desktop, tablet, and mobile dimensions.
- High visual aesthetics using premium curated palettes and fine micro-interactions.
- Seamless mock checkout validation and dynamic price calculators.

3. DESIGN INSPIRATION: INITIAL DESIGN BY STITCH
--------------------------------------------------------------------------------
The visual styling of FreshBite is based on the initial layouts and UI concepts
originally designed by STITCH. 

Stitch's design guidelines focus on evoking warmth, freshness, and high-quality 
culinary comfort. The specific design patterns applied include:
- Warm Appetizing Theme: Replaces generic designs with custom warm cream tones, 
  deep slate chocolate text, and vibrant orange accents to inspire hunger.
- Glassmorphism & Micro-animations: Leverages subtle translucent backdrops, thin 
  borders, shadows, and smooth scaling transitions to make components feel live.
- Typography Pairings: Introduces geometric, modern headings using "Plus Jakarta 
  Sans" and crisp, highly readable text with "Be Vietnam Pro".
- Mobile-First Fluid Grids: Structured from the ground up to collapse columns 
  gracefully on small viewports (e.g., hiding decorative side illustrations on the 
  Sign In page, stacking vertical filter lists, and collapsing the header into a 
  tactile hamburger drawer).

5. LOGICAL ARCHITECTURE & CODE CLEANUP
--------------------------------------------------------------------------------
- State Management: Handled cleanly by two Context files:
  * CartContext: Manages cart state (add, remove, quantity update), coupon 
    discount logic, and wishlist triggers.
  * AuthContext: Simulated login, matching input fields with mock user profiles, 
    and session storage in localStorage.
- Local Storage Integration: Used to save cart counts, addresses, wishlists, and 
  orders so that page refreshes do not erase simulated transactions.
- Clean Code Standard: All files have been optimized to strip extraneous developer 
  notes, debug console statements, and redundant code comments, resulting in a 
  concise, production-ready frontend structure.

6. COMPONENT MATRIX
--------------------------------------------------------------------------------
- Header: Collapsible responsive header with desktop navigation links and mobile 
  hamburger drawer, alongside dynamic cart item indicators.
- CartDrawer: Floating overlay sheet that handles item lists and directly bridges 
  the customer to checkout.
- ProductDetailModal: Implements overlay popups showing ingredients, nutritional 
  information (Calories, Protein, Fat, Carbs), and individual food reviews.
- FeaturedRestaurants: Multi-tab filter grid sorting mock vendors by ratings, 
  delivery time, and culinary specialization.
- Checkout Form: Dynamic forms with validation, express/standard/scheduled shipping
  dropdowns, promo discount applications, and payment loaders.
- Profile & Orders: Dashboard sections providing address books, wishlist arrays, 
  and real-time time-elapsed tracking bars.
